## Directory Scanner

Given directory:
 - List all the entries for all the non-hidden files
 - Nest if a directory has more sub-directory
 - The nesting should be visually pleasing or distinguishing


## How To Run

> `cargo run`

## How To Build

> `cargo build`

_(binary executable is located in `/target`)_
